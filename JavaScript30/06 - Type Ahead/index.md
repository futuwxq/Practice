## 个人总结

+ 网络请求原生 js 
  + WXLHttpsRequest
  + fetch API

+ change 事件
此事件仅限用于`<input>`元素，`<textarea>`和`<select>`元素值的变化

+ 正则表达式的使用
  + match 方法，返回一个数组 或者 null（假值）
  + toLocaleString 方法 ：初始化数字或日期

+ input 等表单事件的 e.key 和 this.value 的区别
  + e.key 是一个键盘的值，一次是一个键盘值
  + this.value 是当前表单值，相对于 e.key 是一个累计值
